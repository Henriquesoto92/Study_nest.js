# Study-Next-JS

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ docker start nest-js-study
$ yarn start

# watch mode
$ yarn dev
$ yarn prisma studio
$ docker start nest-js-study


# production mode
$ yarn start:prod

```
